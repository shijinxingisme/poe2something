# POE2 市场套利分析器

这是一个 POE2 通货市场套利分析 Web 应用。MVP 版本支持输入通货价格和实际交易报价，扫描 A→B→C→A 三步闭环路径，并按收益率展示 Top N 套利机会。

## 核心功能

- 维护通货价格表，价格统一折算为混沌石
- 支持从 poe2db Economy 页面复制通货行并导入价格
- 支持手动录入实际交易报价，例如：`混沌石>崇高石=0.052`
- 扫描 A→B→C→A 闭环路径
- 展示投入、回收、收益率和使用到的三段报价
- 支持设置起点通货、投入数量、Top N 和最低收益率

## 数据来源

行情可以参考：<https://poe2db.tw/cn/Economy>

由于静态网页直接读取第三方页面容易受浏览器限制，MVP 先提供“复制 poe2db 行情文本后导入”的方式。实际套利判断建议以游戏内和交易站即时挂单为准。
