# POE2 国服通货兑换建议器

这是一个用于 POE2 国服通货兑换的网页工具。输入当前市场比例后，它会比较两条兑换神圣石的路线：

- 直接用混沌石兑换神圣石
- 先按崇高石价格折算，再用崇高石兑换神圣石

工具会给出更省的方案、折算成本、预计节省数量、临界比例，以及按你当前库存估算是否够换。

## 使用方式

直接打开 `index.html` 即可使用。也可以把仓库发布到 GitHub Pages 后通过网页访问。

## 输入项

- 需要兑换的神圣石数量
- 1 神圣石对应多少混沌石
- 1 神圣石对应多少崇高石
- 1 崇高石对应多少混沌石
- 可选：已有混沌石和已有崇高石

## 计算原则

所有路线都会先折算成混沌石成本，再比较哪条路线更低。实际交易时仍建议结合挂单数量、成交速度和价格波动判断。
